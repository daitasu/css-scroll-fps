JS なし / CSS のみ

CSS Scroll-Driven
Animations 実験場

animation-timeline: scroll() と view() を使って、JavaScript を一切 書かずにスクロール連動アニメーションを作る作例集。各パターンをクリックして動きを確認してな。

001

ふわっと出現（fade & slide in）

→

要素がビューポートに入るのに合わせて、フェード＋スライド＋スケールで現れる定番の演出。

animation-timeline: view() animation-range
002

固定プログレスバー

→

上部に fixed したバーが、ページのスクロール量に応じて 0 → 100% に伸びる読み進みインジケーター。

animation-timeline: scroll(root) scaleX
003

画像にいろんなアニメ

→

img タグの画像にスクロール連動で回転・拡大・形のモーフ・フィルタ・固定演出をてんこ盛り。

view() view-timeline filter
004

3D 回転キューブ

→

perspective と preserve-3d で組んだ立方体がひたすら回る純 CSS 3D。スクロール連動なし。

perspective transform-style: preserve-3d translateZ
005

FPS フライスルー 🛸

→

CSS 3D × Scroll-Driven の合わせ技。スクロールすると一人称視点で前進し、空に浮かぶ立体の間を飛び抜けていく。

scroll(root) 3D translateZ synthwave
対応ブラウザ: Chrome / Edge 115+、Safari 26+、Firefox（近年版）。未対応環境では静的に表示される フォールバックを各ページに用意してるで。参考: MDN: animation-timeline
